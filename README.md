# 📝 Modern Todo List

A sleek, feature-rich todo list built with vanilla JavaScript, HTML5, and CSS. Glassmorphism design, smooth animations, and persistent local storage.

## 🚀 Live Demo

[Try it here](https://abedoulaye.github.io/todo-list/)

## ✨ Features

- 🎨 Glassmorphism UI with frosted glass effects
- 💾 Tasks save automatically to localStorage
- ✏️ Full CRUD - Create, Read, Update, Delete
- 📱 Responsive on mobile and desktop
- 🛡️ Input validation and XSS protection
- 💬 Warns before discarding unsaved changes

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- Local Storage API
